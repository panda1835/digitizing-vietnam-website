# Digitizing Vietnam Website Design Spec

## Overview
Digitizing Vietnam is a digital archives and research hub exploring the treasure of Vietnamese culture, folk literature, and history. The design style emphasizes historical resonance, readability, and clean layout patterns with natural tones.

## Colors
- **Branding Brown (#a5701c):** Main driver for accents, highlights, active states, and primary buttons.
- **Branding Black (#181818):** Core typography color and primary navigation styling.
- **Branding Gray (#f1f1f1):** Soft background containers and borders.
- **Branding White (#fbfbfb):** Warm canvas base.

## Typography
- **Merriweather (Serif):** Applied to headlines, quotes, and primary titles for a literary feel.
- **Halyard Display (Sans-serif):** Clean sans font for navigational labels, metadata, and controls.

## Layout
- Spacing follows standard Tailwind scale.
- Responsive, mobile-first design with 1200px max content width.

## Shapes
- Large rounded corners (`rounded-3xl` / 24px) for cards.
- Small rounded corners (`rounded-lg` / 8px) for buttons.

## Components
- **Progress Bar:** High contrast green accent to show current progress, rounded container.

## Do's and Don'ts
- Do use branding-brown for primary action indicators and active states.
- Don't use purple or default blue/teal colors.
